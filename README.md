# Tutor-Finder
Tutor-Finder is a Mobile Application that serves the functionality of finding tutor. The system
allows only registered users to login and new users are allowed to register on the application.
It allows the user to find the teacher for subject, for which user is looking for within 3 km of range based on user’s current location. Once the user
finds the tutor, they will be able to communicate for further information via WhatsApp or other
messaging services. There is also a functionality of, if someone wants to teach something. He/She
will register with the app by using “I teach” option with all required details. This application is
location dependent and provide result based on user’s current location.
